# edge-ip-api

基于edge function的ip/地理位置/国家/城市/地区查询api

## 效果
https://ip.api.ravelloh.top/

返回值:  
```json
{
    "ip":"34.29.148.59",
    "city":"Council%20Bluffs",
    "country":"US",
    "latitude":"41.2591",
    "longitude":"-95.8517",
    "region":"sfo1"
}
```
